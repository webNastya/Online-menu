<h2>FRONT-END</h2>
<h4>Установить зависимости</h4>
<code>npm i</code>
<h4>Запустить приложение</h4>
<code>npm run start</code>

</br></br>
Для клиентской части приложения используются:</br>
 - React </br>
 - TypeScript </br>
 - Sass </br>
 - Axios </br>
 - Webpack </br>
 
<h2>BACK-END</h2>
GitHub <a href="https://github.com/Snexok/online-menu-back">online-menu-back</a></br>
Инструкция по развёртыванию серверной части приложения находится в репозитории back-end
</br></br>
Для серверной части приложения используются:</br></br>
 - Nest.JS </br>
 - PostgreSQL </br>
 - TypeORM </br>
 - JWT </br>
